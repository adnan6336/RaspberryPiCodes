
Use the command given below to set up RTC on raspberry pi

wget https://raw.githubusercontent.com/adnan6336/RaspberryPiCodes/main/RTCCode/rtcInstall ; sudo bash rtcInstall

use below command to set time using internet
`sudo python3 /home/pi/DVR/Scripts/setRTCTime.py -i`
to set up manually
`sudo python3 /home/pi/DVR/Scripts/setRTCTime.py`
